# simpledugneon
## a simple online TTRPG written in JavaScript and HTMX served with a side of Bun

### Requirements
- Install [Bun Runtime](https://bun.sh)

### Installation + running 
- To install dependencies: 
```bash 
bun install 
``` 

- To run:
```bash
bun run start 
```

- To run unit tests
```bash
bun test
```
